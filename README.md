#Deepfake Detection via Frequency-Domain Analysis
##Introduction

Recent advances in generative models such as Generative Adversarial Networks (GANs) and diffusion models have significantly improved the realism of synthetic media. While these models produce visually convincing images and videos, they often introduce statistical artifacts in the frequency domain that differ from natural images.

Natural images typically follow a power-law spectral decay, commonly approximated as:

''' E(f) ∝ 1 / f^α '''

where E(f) represents spectral energy at frequency f, and α describes the decay rate. This statistical property emerges from natural scene structure and imaging processes.

In contrast, synthetic images generated by GANs or diffusion models often deviate from this distribution, producing structured artifacts or anisotropic patterns in the Fourier spectrum. These differences have motivated a line of research in frequency-based deepfake detection, where spectral statistics are used as signals to distinguish manipulated media.

This repository contains a set of reproducible experiments designed to analyze spectral properties of real and AI-generated images, and to study how compression affects frequency-domain signals used in deepfake detection.

Experiments

The repository contains several scripts designed to analyze frequency-domain artifacts in real and synthetic media.

1. Frequency Spectrum Analysis

Script:

experiments/frequency_analysis.py

Run:

python experiments/frequency_analysis.py
Purpose

This experiment analyzes Fourier magnitude spectra of images from three categories:

Real images

GAN-generated images

Diffusion-generated images

The script:

Computes the 2D Fourier Transform

Visualizes log-magnitude spectra

Computes radial frequency energy distributions

Output

Results are saved to:

outputs/plots/

Generated figures include:

spectrum comparison grids

radial frequency energy distribution plots

2. Spectral Statistics Analysis

Script:

experiments/spectral_statistics.py

Run:

python experiments/spectral_statistics.py
Purpose

This experiment performs statistical analysis of spectral distributions to quantify differences between real and generated images.

Metrics computed include:

Power-law decay exponent (α) of the radial spectrum

KL divergence between spectral distributions

Statistical significance testing (t-tests)

Output

Saved to:

outputs/statistics/

Generated outputs include:

alpha distribution plots

KL divergence plots

spectral statistics tables

3. Video Compression Sweep

Script:

experiments/compression_sweep.py

Run:

python experiments/compression_sweep.py
Purpose

This experiment evaluates how video compression affects frequency-domain signals.

The script uses FFmpeg to compress a video using H.264 encoding at multiple CRF levels.

Compression levels tested:

CRF = 18, 23, 28, 35, 40

Compressed videos are saved to:

data/video/compressed/
4. Compression Impact Analysis

Script:

experiments/frame_extraction.py

Run:

python experiments/frame_extraction.py
Purpose

This experiment analyzes how compression changes spectral properties of video frames.

The script:

Extracts a frame from the original and compressed videos

Computes Fourier spectra

Measures:

radial frequency distribution

high-frequency energy decay

directional artifact energy

Output

Saved to:

outputs/compression_study/

Generated visualizations include:

FFT comparison grids

radial frequency plots

high-frequency decay curves

directional artifact energy plots

Why These Experiments Matter

Deepfake detection systems often rely on frequency-domain artifacts introduced by generative models. However, these signals may be affected by factors such as:

model architecture

training data biases

image post-processing

video compression

The experiments in this repository provide a transparent analysis of spectral properties in real and synthetic images and help illustrate how these signals behave under realistic conditions.

This work is intended as an exploratory analysis of frequency-domain signals used in deepfake detection, providing insights that may help inform the design of more robust detection systems.
